# 快速开始指南

## 第一步：启动服务器

### 方法一：Python（推荐）
```bash
# 打开终端，进入项目目录
cd "设计心理学实验数据收集网站"

# 运行Python HTTP服务器
python3 -m http.server 8000
```

然后在浏览器中访问：`http://localhost:8000`

### 方法二：直接打开（仅测试用）
直接双击 `index.html` 文件（数据存储在浏览器本地）

## 第二步：配置实验

1. 访问管理员后台：在浏览器中打开 `admin.html`
2. 登录账号：
   - 用户名：`admin`
   - 密码：`admin123`

3. **上传图片素材**（重要！）：
   - 点击"素材管理"标签
   - 上传色调组图片（8张）
   - 上传明度组图片（12张）
   - 上传饱和度组图片（12张）

4. **配置实验参数**（可选）：
   - 点击"实验配置"标签
   - 检查人格特质形容词（默认已配置）
   - 检查情境描述（默认："首次加入新社群"）
   - 调整其他参数（如需要）
   - 点击"保存配置"

## 第三步：开始实验

1. 在浏览器中打开 `index.html` 或访问 `http://localhost:8000`
2. 被试按照界面提示完成实验
3. 数据自动保存

## 第四步：导出数据

1. 返回管理员后台 `admin.html`
2. 点击"数据管理"标签
3. 选择日期范围（可选）
4. 点击"导出数据（CSV）"或"导出数据（Excel）"
5. 文件会自动下载

## 注意事项

- ⚠️ **首次使用必须先上传图片素材**，否则实验无法进行
- ⚠️ 数据存储在浏览器localStorage中，清空浏览器数据会丢失所有数据
- 💡 建议定期导出数据并备份
- 💡 图片建议大小不超过2MB，确保加载速度

## 常见问题

**Q: 实验界面显示"未找到实验图片"？**
A: 需要先在管理员后台上传图片素材。

**Q: 如何修改管理员密码？**
A: 修改 `js/config.js` 文件中的 `admin` 对象，或联系开发者添加修改密码功能。

**Q: 数据在哪里？**
A: 数据存储在浏览器的localStorage中，使用管理员后台的"数据管理"功能可以导出。

更多详细信息请查看 `README.md` 文件。
